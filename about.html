About Me

Hey, I’m Muzziey — a creative software engineer student, web designer, programmer, and part-time skateboarder who loves turning ideas into functional code. I specialize in Python, HTML, C++, JavaScript, and MySQL, mostly working inside Visual Studio Code and GitHub.

Whether I’m designing sleek user interfaces or developing systems behind the scenes, I believe in one core principle:

"Design for Change."

Great software isn’t just about how it works today — it’s about how easily it can evolve tomorrow. I build with flexibility in mind, creating systems with "seams" that welcome future change rather than resist it.

When I'm not coding or designing, you’ll probably find me on a skateboard, chasing the same flow and balance I aim for in my work.

Let’s build something meaningful — and maybe even a little disruptive.
